Barrels and Backpacks

By Preston Anderson

This is intended to be the backend code for the inventory systems, however eventually I will make this visual and forego the whole console situation.
Initially I had planned for the whole loop of the project to be different by going through different steps of adding, removing, and moving items through each inventory,
but I found it easier to make it a menu for your options instead.

This project is now a project for fun, as this will no longer be continued. The visual version of this project will come some day.
It works perfectly as is, and should be able to be edited to add anything further if necessary without breaking the code.

Thank you for using this janky inventory.
